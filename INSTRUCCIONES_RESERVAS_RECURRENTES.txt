📝 CÓMO USAR LAS NUEVAS FUNCIONES:
───────────────────────────────────────────────────────────────────────

1️⃣  DÍAS RECURRENTES (ej: "Todos los martes cerrados")

   ► Abre la app → Configuración → Reservas → Tab "Vacaciones"
   ► Toca el botón naranja "Recurrente" 🟠
   ► Marca los días que quieres cerrar siempre (ej: Martes, Domingo)
   ► Toca "Guardar"
   ► ✅ Aparecerán chips naranjas con los días seleccionados
   ► Toca "Guardar" arriba para sincronizar con web


2️⃣  INTERVALOS DE FECHAS (ej: "Del 13 al 15 mayo - Vacaciones")

   ► En el mismo Tab "Vacaciones"
   ► Toca el botón morado "Intervalo" 🟣
   ► "Desde..." → Selecciona fecha inicio (ej: 13 mayo)
   ► "Hasta..." → Selecciona fecha fin (ej: 15 mayo)
   ► Escribe motivo: "Vacaciones de verano" (opcional)
   ► Toca "Guardar"
   ► ✅ Aparecerá una tarjeta morada con el rango
   ► Toca "Guardar" arriba para sincronizar


3️⃣  TIME PICKER ARREGLADO (cambiar hora de apertura/cierre)

   ► Tab "Horarios"
   ► Toca el chip verde "09:00" (hora de apertura)
   ► Ahora puedes deslizar el dedo arriba/abajo
   ► ✅ Las horas cambian correctamente (9 → 10 → 11...)
   ► Confirmar

💡 EJEMPLOS DE USO:
───────────────────────────────────────────────────────────────────────

🍕 RESTAURANTE:
   "Cerramos todos los lunes"
   → Botón Recurrente → Marcar Lunes → Guardar

🏖️ HOTEL:
   "Cerrado del 1 al 31 de agosto por vacaciones"
   → Botón Intervalo → Desde: 1 ago → Hasta: 31 ago
   → Motivo: "Vacaciones de verano" → Guardar

💇 PELUQUERÍA:
   "Domingos cerrados + festivos navideños (24-26 dic)"
   → Recurrente: Domingo
   → Intervalo: 24 dic - 26 dic, motivo: "Navidad"
   → Día específico: 1 enero, motivo: "Año nuevo"


🔄 SINCRONIZACIÓN:
───────────────────────────────────────────────────────────────────────

Cuando tocas "Guardar" arriba, la app:

1. Guarda en Firestore: empresas/{id}/configuracion/reservas
2. Sincroniza con:  empresas/{id}/configuracion/reservas_web
3. El formulario web lee de reservas_web automáticamente

⚠️ IMPORTANTE: Siempre toca "Guardar" después de hacer cambios


🐛 SI ALGO NO FUNCIONA:
───────────────────────────────────────────────────────────────────────

❌ "No puedo mover el selector de horas"
   → Cierra y vuelve a abrir la app
   → Si persiste: flutter clean && flutter run

❌ "Los días recurrentes no aparecen en web"
   → Verifica que hayas tocado "Guardar" en la app
   → Refresca el formulario web (F5)
   → Verifica en Firebase Console que existen en reservas_web

❌ "El intervalo no bloquea reservas"
   → Verifica que las fechas estén bien (inicio < fin)
   → Guarda de nuevo la configuración
   → El formulario web debe leer intervalos_cerrados
