# Sonidos de Notificación

Coloca aquí los siguientes archivos MP3:

| Archivo             | Descripción                                  |
|---------------------|----------------------------------------------|
| notif_default.mp3   | Sonido por defecto                           |
| notif_urgente.mp3   | Sonido urgente/alerta                        |
| notif_suave.mp3     | Sonido suave/tranquilo                       |
| notif_digital.mp3   | Sonido digital/tech                          |
| notif_clasico.mp3   | Sonido clásico de notificación               |
| sin_sonido.mp3      | Archivo de silencio (0.1 seg)                |

## Requisitos
- Formato: MP3
- Duración recomendada: 1-3 segundos (excepto sin_sonido.mp3 que debe ser ~0.1s)
- Calidad: 128kbps mínimo

## Fuentes gratuitas
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org
- https://notificationsounds.com
