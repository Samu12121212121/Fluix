# Icono de la App

Coloca aquí el archivo:

| Archivo         | Requisitos                                                          |
|-----------------|---------------------------------------------------------------------|
| app_icon.png    | 1024×1024px, PNG, sin esquinas redondeadas (el SO las aplica), fondo de color de marca o transparente |

## Generar iconos tras colocar el archivo

```bash
flutter pub run flutter_launcher_icons
```
