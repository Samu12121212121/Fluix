 EL PROBLEMA

Firebase SDK para Windows tiene:
  cmake_minimum_required(VERSION 2.8.12)

CMake moderno (tu versión) requiere:
  Mínimo VERSION 3.10

❌ NO SE PUEDE EVITAR desde CMakeLists.txt del proyecto
❌ Las políticas antiguas (CMP0000) ya no funcionan
❌ No hay configuración que lo solucione

✅ LA UNICA SOLUCIÓN QUE FUNCIONA
