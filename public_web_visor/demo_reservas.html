📅 Widget de Reservas

Demo del widget embebible — powered by Fluix CRM

Cómo integrar en tu web

<!-- 1. Div contenedor donde quieres que aparezca el widget --> <div id="fluix-reservas"></div> <!-- 2. Configuración (antes del script) --> <script> window.FluixReservas = { empresaId: 'TU_EMPRESA_ID', // ← ID de tu empresa en Firestore titulo: 'Reserva tu cita', color: '#1976D2', // ← Color de tu marca horaInicio: 9, // ← Hora apertura (24h) horaFin: 20, // ← Hora cierre (24h) intervaloMin: 30, // ← Slots cada N minutos diasSemana: [1,2,3,4,5], // ← 0=Dom, 1=Lun … 6=Sáb }; </script> <!-- 3. Script del widget --> <script src="reservas.js" defer></script>
